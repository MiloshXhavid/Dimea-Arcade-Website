  <name>Task 2: Replace JS — implement JSONP submit, replace handleHwWaitlist(), add handleFooterNewsletter()</name>
  <files>index.html</files>
  <action>
    Replace the existing `handleHwWaitlist()` function in the inline script block (line ~1863) and add the shared JSONP helper and footer handler. All JS stays in the single inline &lt;script&gt; block at the bottom of &lt;body&gt;.

    PREREQUISITE: Read the MC_ACTION_URL, HW_GROUP_PARAM, NL_GROUP_PARAM values from 09-01-SUMMARY.md before writing code. Use the real values Dimitri provided. If placeholders remain (e.g. [ACTION_URL_FROM_DIMITRI]), keep them as clearly marked TODO comments.

    REPLACE the existing handleHwWaitlist() block (lines ~1863–1875) with:

    ```javascript
    // ===================== MAILCHIMP =====================
    var MC_ACTION_URL = 'PASTE_ACTION_URL_FROM_09-01-SUMMARY'; // e.g. https://dimea.us1.list-manage.com/subscribe/post?u=XXX&id=YYY
    var HW_GROUP_PARAM = 'PASTE_HW_GROUP_PARAM_FROM_09-01-SUMMARY'; // e.g. &group[01234][1]=1
    var NL_GROUP_PARAM = 'PASTE_NL_GROUP_PARAM_FROM_09-01-SUMMARY'; // e.g. &group[01234][2]=1

    function submitToMailchimp(email, groupParam, onSuccess, onError) {
      var callbackName = 'mcCb_' + Date.now();
      var url = MC_ACTION_URL.replace('/post?', '/post-json?');
      url += '&EMAIL=' + encodeURIComponent(email);
      url += groupParam;
      url += '&c=' + callbackName;

      window[callbackName] = function(data) {
        delete window[callbackName];
        if (data.result === 'success') {
          onSuccess();
        } else {
          // "already subscribed" is not an error — subscriber is valid
          if (data.msg && data.msg.indexOf('already subscribed') !== -1) {
            onSuccess();
          } else {
            onError(data.msg || 'Something went wrong — try again');
          }
        }
      };

      var script = document.createElement('script');
      script.src = url;
      document.head.appendChild(script);
      script.onload = function() { this.remove(); };
    }

    function handleHwWaitlist() {
      var input = document.getElementById('hwEmailInput');
      var btn = document.querySelector('.hw-email-btn');
      var confirmEl = document.getElementById('hwConfirm');
      var errorEl = document.getElementById('hwError');

      // Reset error state
      errorEl.style.display = 'none';
      errorEl.textContent = '';

      if (!input.value || !input.value.includes('@')) {
        input.style.borderColor = 'rgba(232,58,110,0.6)';
        setTimeout(function() { input.style.borderColor = ''; }, 800);
        return;
      }

      input.disabled = true;
      btn.disabled = true;

      submitToMailchimp(input.value, HW_GROUP_PARAM,
        function() {
          btn.textContent = 'Done ✓';
          confirmEl.classList.add('visible');
        },
        function(msg) {
          input.disabled = false;
          btn.disabled = false;
          errorEl.textContent = msg;
          errorEl.style.display = 'block';
        }
      );
    }

    function handleFooterNewsletter() {
      var input = document.querySelector('.footer-email-input');
      var btn = document.querySelector('.footer-email-btn');
      var confirmEl = document.getElementById('footerConfirm');
      var errorEl = document.getElementById('footerError');

      // Reset error state
      errorEl.style.display = 'none';
      errorEl.textContent = '';

      if (!input.value || !input.value.includes('@')) {
        input.style.borderColor = 'rgba(232,58,110,0.6)';
        setTimeout(function() { input.style.borderColor = ''; }, 800);
        return;
      }

      input.disabled = true;
      btn.disabled = true;

      submitToMailchimp(input.value, NL_GROUP_PARAM,
        function() {
          btn.textContent = '✓';
          confirmEl.style.display = 'block';
        },
        function(msg) {
          input.disabled = false;
          btn.disabled = false;
          errorEl.textContent = msg;
          errorEl.style.display = 'block';
        }
      );
    }
    // ===================== END MAILCHIMP =====================
    ```

    CRITICAL IMPLEMENTATION NOTES:
    - Use `var` not `const`/`let` — consistent with adjacent existing JS in this block (filterImp uses const but the wider pattern is mixed; var is safer for the global MC_ constants)
    - Do NOT use fetch() — Mailchimp's endpoint has no CORS headers. JSONP script tag injection is the only approach that allows reading the response.
    - The `callbackName` uses `Date.now()` suffix to prevent collision if both forms submit simultaneously
    - `script.onload` removes the injected script tag from the DOM to keep it clean
    - The `already subscribed` branch treats returning subscribers as a success — prevents confusing error messages for people who signed up previously
    - Button disable during submit prevents double-submit. Re-enabled only on error path (success path leaves it disabled permanently as intended)

    After replacing, verify the old handleHwWaitlist() stub is fully gone (no duplicate function definition). The // ===================== CURSOR ===================== comment block should still follow immediately after the new Mailchimp block.
